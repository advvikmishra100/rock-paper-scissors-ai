rock-paper-scissors.ai
🧠 How the AI Works
The “Neural” aspect of this project is powered by a Frequency Analysis Algorithm.

Data Collection: The script tracks every move the player makes (Rock, Paper, or Scissors).
Pattern Analysis: It calculates which move the player chooses most frequently.
Counter-Strategy:
If the player favors Rock, the AI plays Paper.
If the player favors Paper, the AI plays Scissors.
If the player favors Scissors, the AI plays Rock.
Initial Randomness: To prevent immediate predictability, the AI plays randomly for the first 3 rounds to gather initial data.
✨ Features
Minimalist UI: Built with a modern “Glassmorphism” design using CSS backdrop filters.
Responsive Design: Works on desktops and mobile browsers.
Dynamic Feedback: Real-time score tracking and color-coded win/loss indicators.
Lightweight: Zero dependencies. Just vanilla HTML, CSS, and JavaScript.
